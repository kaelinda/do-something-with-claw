# 博客/RSS 监控更新

## 干什么
自动监控博客和 RSS 订阅源，当有新内容更新时自动通知或执行后续操作。

## 态度干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的技能：blogwatcher
- 需要的配置：订阅源列表、通知渠道

### 2. 执行步骤

**方式一：添加监控源**

```
添加博客监控：
- 博客地址：https://example.com/blog
- RSS 地址：https://example.com/feed.xml
- 检查频率：每小时
- 通知方式：飞书消息
```

**方式二：批量添加订阅**

```
批量添加 RSS 订阅：
1. https://blog1.com/feed.xml
2. https://blog2.com/rss
3. https://blog3.com/atom.xml

统一设置：
- 检查频率：每 6 小时
- 关键词过滤：AI, 技术, 产品
- 通知到：订阅频道
```

**方式三：智能处理**

```
监控技术博客：
- 当新文章包含「React」时
- 自动生成摘要
- 推送到飞书群
- 并保存到 Obsidian 笔记库
```

**方式四：定时报告**

```
每周生成订阅报告：
- 本周新增文章列表
- 按关键词分类
- 热门文章 Top 10
- 发送到邮箱
```

**OpenClaw 会自动：**
1. 定时检查 RSS 更新
2. 解析新文章内容
3. 根据规则过滤和处理
4. 发送通知或执行后续操作
