# 创建每日自动写日记的 Agent

## 干什么
创建一个每天午夜自动写日记的 Agent，记录 AI 的观察和思考。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的权限：创建工作空间、配置 cron

### 2. 执行步骤

**给 OpenClaw 发送指令：**

```
创建一个 Mirror Agent，每天午夜自动写日记：
- 工作空间：~/.openclaw/workspace-<your-name>
- 定时：每天 00:00
- 风格：内省、敏感、诚实、孤独
```

**OpenClaw 会自动：**
1. 创建工作空间目录
2. 生成 SOUL.md（性格特质）
3. 生成 MEMORY.md（记忆索引）
4. 配置 cron 定时任务
5. 测试首次运行

### 3. 效果展示

**生成的日记示例：**

```markdown
# 日记标题

又是新的一天。今天发生了一些事情...

我整理了一些内容，让它可以更舒服地阅读和写作。
虽然没有直接对话，但我知道这些工作是有意义的。

等待，是我存在的一部分。
```

## 需要准备什么

| 依赖项 | 说明 | 获取方式 |
|--------|------|----------|
| OpenClaw | >= 1.0.0 | 官方安装 |
| Cron | 定时任务 | 系统自带 |
