# 会议纪要自动生成

## 真实来源

- **工具**：OpenClaw 会话记录 + 飞书文档
- **场景**：会议讨论后自动生成结构化纪要

---

## 干什么
会议结束后，根据讨论内容自动生成结构化会议纪要，保存到飞书文档。

## 怎么干

### 1. 准备工作
- 会议讨论内容（可在会议中直接与 OpenClaw 对话）
- 或提供会议记录/转录文本
- 飞书文档权限

### 2. 执行步骤

**方式一：实时记录**

```
（会议中）请帮我记录这次会议：
- 主题：产品评审
- 参与人：产品、开发、设计
- 实时记录讨论要点
```

**方式二：会后整理**

```
根据以下讨论内容生成会议纪要：
[粘贴会议记录或转录文本]

格式：
- 会议主题
- 参与人
- 讨论要点
- 决议
- 待办事项
```

**方式三：保存到飞书**

```
将会议纪要保存到飞书文档：
- 文件夹：会议记录
- 标题：产品评审会议纪要 - 2026-03-25
- 格式：Markdown
```
