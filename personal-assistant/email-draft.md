# 邮件草稿生成

## 干什么
根据需求快速生成专业的邮件草稿，支持多种场景和语气风格。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的技能：无特殊要求
- 需要的配置：无

### 2. 执行步骤

**给 OpenClaw 发送指令：**

```
帮我写一封邮件：
- 收件人：技术总监
- 主题：项目延期预警
- 内容要点：
  1. 当前进度比计划落后 3 天
  2. 原因：第三方 API 不稳定
  3. 需要支持：协调资源处理接口问题
- 语气：专业、客观、有解决方案
```

**OpenClaw 会自动：**
1. 理解邮件场景和收件人
2. 采用合适的语气和格式
3. 生成完整邮件草稿
4. 支持多轮修改调整

### 3. 效果展示

```markdown
主题：【项目预警】用户中心重构项目进度同步

王总您好，

用户中心重构项目当前进度比原计划落后 3 天，特此同步情况并申请支持。

**当前状态**
- 已完成：用户注册/登录模块（100%）
- 进行中：权限管理模块（60%）
- 待启动：消息中心模块

**延期原因**
集成第三方短信服务时遇到 API 不稳定问题，平均响应时间超过 5 秒，
已影响登录流程的用户体验。目前已尝试 3 次优化，效果有限。

**解决方案**
1. 短期：切换到备用短信服务商（需采购审批）
2. 长期：自建短信网关（预计 2 周）

**需要支持**
- 请帮忙协调采购部门加速审批流程
- 建议本周三组织跨部门会议讨论方案

预计调整后，项目可延期 2 天完成。如需更多信息，随时沟通。

祝好，
张三
```
