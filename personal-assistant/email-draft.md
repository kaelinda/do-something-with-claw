# 邮件草稿生成

## 真实来源

- **工具**：OpenClaw 对话生成
- **辅助**：himalaya 技能（发送邮件）
- **场景**：商务邮件、通知邮件、汇报邮件

---

## 干什么
根据需求快速生成专业的邮件草稿，支持多种场景和语气风格。

## 怎么干

### 1. 准备工作

- OpenClaw 版本：>= 1.0.0
- 邮件场景信息

### 2. 执行步骤

**方式一：商务汇报邮件**

```
帮我写一封邮件：
- 收件人：技术总监
- 主题：项目延期预警
- 内容要点：
  1. 当前进度比计划落后 3 天
  2. 原因：第三方 API 不稳定
  3. 需要支持：协调资源处理接口问题
- 语气：专业、客观、有解决方案
```

**方式二：会议邀请邮件**

```
写一封会议邀请邮件：
- 会议主题：产品需求评审
- 时间：本周四 14:00-16:00
- 地点：会议室 A
- 参与人：产品、开发、设计团队
- 附件：需求文档链接
```

**方式三：客户沟通邮件**

```
写一封客户回复邮件：
- 客户问题：产品功能咨询
- 问题详情：[paste customer question]
- 回复要点：功能介绍、使用方法、联系方式
- 语气：友好、专业
```

**方式四：周报邮件**

```
生成本周工作周报邮件：
- 收件人：直属领导
- 本周完成：
  1. 完成用户登录模块开发
  2. 修复 5 个 bug
  3. 参加产品评审会议
- 下周计划：
  1. 开始权限模块开发
  2. 编写单元测试
- 需要支持：无
```

### 3. 邮件模板

**商务汇报邮件**：
```markdown
主题：【项目预警】用户中心重构项目进度同步

王总您好，

用户中心重构项目当前进度比原计划落后 3 天，特此同步情况并申请支持。

**当前状态**
- 已完成：用户注册/登录模块（100%）
- 进行中：权限管理模块（60%）
- 待启动：消息中心模块

**延期原因**
集成第三方短信服务时遇到 API 不稳定问题，平均响应时间超过 5 秒。

**解决方案**
1. 短期：切换到备用短信服务商（需采购审批）
2. 长期：自建短信网关（预计 2 周）

**需要支持**
- 请帮忙协调采购部门加速审批流程
- 建议本周三组织跨部门会议讨论方案

预计调整后，项目可延期 2 天完成。

祝好，
张三
```

**周报邮件**：
```markdown
主题：工作周报 - 张三 - 2026-03-25

李总您好，

以下是本周工作汇报：

**本周完成**
1. ✅ 完成用户登录模块开发
2. ✅ 修复 5 个 bug（#123, #124, #125, #126, #127）
3. ✅ 参加产品评审会议

**下周计划**
1. 开始权限模块开发
2. 编写单元测试
3. 参与技术分享会

**需要支持**
暂无

**其他**
新版本已上线，目前运行稳定。

祝好，
张三
```
