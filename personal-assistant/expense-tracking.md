# 记账助手

## 真实来源

- **技能**：feishu_bitable（飞书多维表格）、obsidian
- **工具**：exec + OpenClaw 对话能力
- **技能位置**：
  - 飞书多维表格：内置工具 `feishu_bitable_*`
  - Obsidian：`/opt/homebrew/lib/node_modules/openclaw/skills/obsidian/`

---

## 干什么
使用 OpenClaw 进行日常记账，自动记录支出、生成财务报告、分析消费习惯。支持飞书多维表格和 Obsidian 两种存储方案。

## 怎么干

### 1. 准备工作

- OpenClaw 版本：>= 1.0.0
- 存储方案：飞书多维表格 或 Obsidian

### 2. 执行步骤

#### 方式一：对话式记账
```
用户：今天午饭花了 35 元
OpenClaw：✅ 已记录支出
- 日期：2024-01-15
- 类别：餐饮
- 金额：35 元
- 备注：午饭

本月餐饮支出：1,280 元
本月总支出：3,560 元
```

#### 方式二：飞书多维表格记账

**创建记账表格：**

```
帮我在飞书创建一个记账多维表格，包含以下字段：
- 日期
- 类型（收入/支出）
- 金额
- 类别
- 账户
- 备注
```

**使用 feishu_bitable_create_record 记录：**

```javascript
// OpenClaw 自动调用
feishu_bitable_create_record({
  app_token: "xxx",
  table_id: "tblxxx",
  fields: {
    "日期": 1705276800000,  // timestamp
    "类型": "支出",
    "金额": 35,
    "类别": "餐饮",
    "账户": "支付宝",
    "备注": "午饭"
  }
})
```

#### 方式三：Obsidian 记账

**创建记账笔记：**

```
在 Obsidian 中创建今日账单：
- 午餐 35 元 #餐饮 #支付宝
- 地铁 6 元 #交通 #微信
```

**Obsidian 格式：**

```markdown
# 2024-01 账单

## 2024-01-15
- 午餐 35 元 #餐饮 #支付宝
- 地铁 6 元 #交通 #微信
- 咖啡 28 元 #餐饮 #支付宝
- 晚餐 45 元 #餐饮 #微信

## 本月统计
- 总支出：3,560 元
- 总收入：15,000 元
```

## 数据存储方案

### 飞书多维表格

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 交易日期 |
| 类型 | 单选 | 收入/支出 |
| 金额 | 数字 | 交易金额 |
| 类别 | 单选 | 餐饮/交通/购物等 |
| 账户 | 单选 | 支付宝/微信/现金等 |
| 备注 | 文本 | 详细说明 |

### Obsidian 方案

```markdown
# 2024-01 账单

## 2024-01-15
- 午餐 35 元 #餐饮 #支付宝
- 地铁 6 元 #交通 #微信
- 咖啡 28 元 #餐饮 #支付宝
- 晚餐 45 元 #餐饮 #微信

## 本月统计
- 总支出：3,560 元
- 总收入：15,000 元
```
