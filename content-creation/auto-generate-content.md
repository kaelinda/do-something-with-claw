# 自动生成自媒体内容

## 干什么
让 OpenClaw 帮你生成高质量的自媒体内容，支持微信公众号、小红书、抖音文案等平台风格。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的技能：`viral-writer` 技能
- 需要的配置：无

### 2. 执行步骤

**给 OpenClaw 发送指令：**

```
帮我写一篇小红书笔记，主题是"AI 助手如何改变我的工作效率"：
- 风格：小红书 emoji 风格
- 字数：1000 字以内
- 包含标题建议
```

**OpenClaw 会自动：**
1. 分析主题和目标受众
2. 生成多个标题选项
3. 撰写正文内容
4. 添加 emoji 和排版
5. 生成配图建议

### 3. 效果展示

**生成的标题选项：**
1. 🔥 用 AI 助手 3 个月，我的效率翻倍了
2. 每天省 2 小时！AI 助手真的香
3. 打工人必备：让 AI 帮你干活的正确姿势

**正文预览：**
```
姐妹们！今天来分享一个让我效率翻倍的神器...

用了 3 个月，我的日常：
✅ 自动整理会议纪要
✅ 帮我写周报
✅ 甚至还能帮我审代码...

重点是——只要说一句话就行！
...
```
