# 视频脚本生成

## 真实来源

- **工具**：OpenClaw 对话生成
- **辅助**：gemini 技能、飞书文档
- **TTS**：内置 tts 工具

---

## 干什么
使用 OpenClaw 自动生成视频脚本，包括短视频、教学视频、产品解说等多种类型。

## 怎么干

### 1. 准备工作

- OpenClaw 版本：>= 1.0.0
- 可选：gemini 技能、飞书文档

### 2. 执行步骤

**方式一：短视频脚本**

```
帮我写一个 60 秒的抖音短视频脚本：

主题：如何用 AI 提高工作效率
风格：快节奏、干货型
开场：3 秒抓住眼球
结尾：引导关注
```

**方式二：教学视频脚本**

```
生成一个 10 分钟的教学视频脚本：

主题：Python 入门第一课
目标观众：零基础学习者
结构：
1. 开场引入
2. 环境准备
3. 基础语法
4. 实操演示
5. 总结回顾
```

**方式三：产品解说脚本**

```
为这款产品写一个 3 分钟的解说脚本：

产品：智能日程管理 App
核心功能：
- AI 日程规划
- 会议提醒
- 任务追踪

目标用户：职场人士
风格：专业、简洁
```

**方式四：配音生成**

```
为这段脚本生成配音：

脚本内容：
[paste script]

配音要求：
- 声音：专业男声
- 语速：适中
- 情感：积极向上
```

### 3. 脚本结构模板

**短视频脚本**：
```markdown
# 视频标题

时长：60 秒

## 开场（0-3 秒）
[画面：吸引眼球的画面]
旁白：你还在手动整理数据？

## 痛点（3-15 秒）
[画面：问题场景]
旁白：每天花费大量时间...

## 方案（15-45 秒）
[画面：解决方案演示]
旁白：试试这个 AI 工具...

## 结尾（45-60 秒）
[画面：成果展示 + CTA]
旁白：点击关注，获取更多技巧！
```

**教学视频脚本**：
```markdown
# 课程标题

时长：10 分钟
目标：学习目标说明

## 开场（1 分钟）
- 自我介绍
- 课程价值
- 学习目标

## 准备工作（2 分钟）
- 工具准备
- 前置知识

## 核心内容（5 分钟）
### 模块 1
- 概念讲解
- 代码演示

### 模块 2
- 进阶内容
- 实例演示

## 总结（2 分钟）
- 知识点回顾
- 练习任务
- 下期预告
```

```
视频脚本生成完成！
```
