# SEO 优化内容

## 真实来源

- **工具**：OpenClaw 对话生成
- **辅助**：gemini 技能、web_search 工具
- **场景**：SEO 内容优化、关键词研究

---

## 干什么
让 OpenClaw 帮你生成 SEO 友好的内容，包括关键词优化、元描述、标题优化、内容结构化。

## 怎么干

### 1. 准备工作

- OpenClaw 版本：>= 1.0.0
- 目标关键词列表

### 2. 执行步骤

**方式一：生成 SEO 标题和描述**

```
为以下文章生成 SEO 优化内容：

主题：AI 助手提升工作效率
目标关键词：AI 助手、效率提升、自动化工作流

生成：
1. 5 个标题选项（包含关键词）
2. Meta 描述（150-160 字符）
3. H2/H3 大纲建议
```

**方式二：内容 SEO 优化**

```
优化以下内容的 SEO：

原文：
[paste your content]

目标关键词：OpenClaw、自动化、AI 助手
要求：
- 关键词密度 2-3%
- 自然融入关键词
- 添加相关长尾词
```

**方式三：关键词研究**

```
帮我研究关键词"AI 助手"：
- 相关长尾关键词
- 搜索意图分析
- 内容建议
```

**方式四：批量生成 SEO 内容**

```
为以下关键词批量生成 SEO 文章大纲：

关键词列表：
1. 自动化工作流
2. AI 生产力工具
3. 智能助手应用

每个关键词生成：
- 标题
- Meta 描述
- 文章大纲（H2/H3）
```

### 3. SEO 输出模板

**标题选项**：
```markdown
1. AI 助手完全指南：如何用 OpenClaw 实现 10 倍效率提升
2. 2024 年最佳 AI 助手工具：自动化工作流实战手册
3. AI 助手入门教程：从零开始打造自动化工作流
4. 智能助手 vs 传统工具：AI 助手如何改变你的工作方式
5. 效率翻倍！用 AI 助手构建自动化工作流的 5 个秘诀
```

**Meta 描述**：
```markdown
探索 AI 助手如何通过自动化工作流提升工作效率。本指南详解 OpenClaw 
的核心功能、实用技巧和真实案例，助你快速上手智能助手。
```

**内容大纲**：
```markdown
# AI 助手完全指南

## 什么是 AI 助手？（H2）
### AI 助手的定义（H3）
### AI 助手的工作原理（H3）

## 为什么需要 AI 助手？（H2）
### 效率提升的 3 大理由（H3）
### 真实案例分享（H3）

## 如何选择 AI 助手？（H2）
### 核心功能对比（H3）
### 价格与性价比（H3）
```

### 4. 效果展示

```
SEO 优化完成！

关键词分析：
- 主关键词：AI 助手
- 搜索量：约 10,000/月
- 竞争度：中等
- 相关长尾词：15 个

标题优化：
- 原标题：如何使用 AI 助手
- 优化后：AI 助手完全指南：2024 年效率提升实战手册
- 评分：A（关键词前置、数字吸引、年份时效）

Meta 描述：
- 字符数：156（符合 150-160 最佳范围）
- 关键词包含：AI 助手、效率提升、自动化
- CTA：助你快速上手

内容优化建议：
✅ H1 标题包含主关键词
✅ 前 100 字出现主关键词
✅ H2/H3 结构化层级
⚠️ 建议：添加更多内链
⚠️ 建议：增加图片 alt 属性
```
