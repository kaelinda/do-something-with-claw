# 播客脚本生成

## 真实来源

- **工具**：OpenClaw 对话生成
- **辅助**：gemini 技能、飞书文档
- **TTS**：内置 tts 工具

---

## 干什么
使用 OpenClaw 自动生成播客脚本，支持单口播客、访谈播客、专题播客等多种形式。

## 怎么干

### 1. 准备工作

- OpenClaw 版本：>= 1.0.0
- 可选：gemini 技能、飞书文档

### 2. 执行步骤

**方式一：单口播客**

```
帮我写一期 20 分钟的单口播客脚本：

主题：AI 如何改变工作方式
风格：轻松对话式
结构：
1. 开场白（2 分钟）
2. 背景引入（3 分钟）
3. 核心观点（12 分钟）
4. 总结收尾（3 分钟）
```

**方式二：访谈播客**

```
生成一期访谈播客脚本：

嘉宾：技术专家
主题：程序员的职业发展
时长：30 分钟
环节：
- 嘉宾介绍（3 分钟）
- 背景了解（5 分钟）
- 核心话题（15 分钟）
- 快问快答（4 分钟）
- 总结收尾（3 分钟）
```

**方式三：专题系列**

```
规划一个 5 集的播客系列：

主题：创业者访谈
每集时长：30 分钟
目标听众：创业者、职场人士
要求：
- 每集一个核心话题
- 系列连贯性
- 下期预告
```

**方式四：配音预览**

```
为播客脚本生成配音预览：

脚本：[paste script]
配音风格：
- 声音：专业男声
- 语速：适中
- 情感：轻松友好
```

### 3. 脚本模板

**单口播客**：
```markdown
# 播客标题

时长：20 分钟
风格：轻松对话

## 开场白（2 分钟）
大家好，欢迎来到今天的节目！
我是主持人 [名字]...
今天我们要聊一个很有意思的话题...

## 背景引入（3 分钟）
最近，[热点事件]让所有人都在讨论...
我想和大家分享我的看法...

## 核心观点（12 分钟）
### 观点 1（4 分钟）
首先，我想说的是...
举个例子...

### 观点 2（4 分钟）
其次，我认为...
数据表明...

### 观点 3（4 分钟）
最后，还有一点很重要...

## 总结收尾（3 分钟）
总结一下今天的观点...
下期节目我们将聊...
感谢收听，我们下期再见！
```

**访谈播客**：
```markdown
# 访谈播客脚本

嘉宾：[姓名]
主题：[话题]
时长：30 分钟

## 开场（3 分钟）
主持人：欢迎来到本期节目...
今天我们邀请到了 [嘉宾名]...
[嘉宾介绍]

## 背景了解（5 分钟）
Q1: 请简单介绍一下自己...
Q2: 您是如何进入这个领域的...

## 核心话题（15 分钟）
Q3: 关于 [主题]，您的看法是...
Q4: 能分享一个具体案例吗...
Q5: 遇到的最大挑战是...
Q6: 您认为未来趋势是...

## 快问快答（4 分钟）
- 最近在读什么书？
- 最推荐的一个工具？
- 一句话建议...

## 结尾（3 分钟）
嘉宾：总结发言...
主持人：感谢嘉宾，下期预告...
```
