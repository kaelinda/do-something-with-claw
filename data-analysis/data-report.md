# 数据报告生成

## 干什么
自动分析数据源并生成结构化数据报告，支持多种数据格式和可视化输出。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的技能：无特殊要求
- 需要的配置：数据源访问权限

### 2. 执行步骤

**方式一：从 CSV/Excel 分析**

```
分析这份销售数据：
~/Documents/data/sales-2026-Q1.csv

生成报告：
1. 销售趋势分析
2. Top 10 产品
3. 区域分布
4. 环比增长率
```

**方式二：从数据库提取**

```
生成用户增长报告：
- 数据源：MySQL users 表
- 时间范围：最近 30 天
- 指标：新增用户、活跃用户、留存率
```

**方式三：从 API 获取**

```
从飞书多维表格生成报告：
- 表格：[飞书多维表格链接]
- 分析字段：需求状态、优先级分布
- 输出：需求进展报告
```

**方式四：定时报告**

```
设置每周一 9:00 自动生成周报：
- 数据源：GitHub Issues
- 范围：上周创建/关闭的 Issue
- 发送到：飞书群
```

**OpenClaw 会自动：**
1. 读取/获取数据
2. 执行数据分析
3. 生成可视化图表（文本描述）
4. 输出结构化报告

