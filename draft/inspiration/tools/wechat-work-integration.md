# 企业微信操作

## 干什么
让 OpenClaw 自动发送企业微信消息，包括文本、卡片、文件，支持群聊和私聊。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的配置：企业微信机器人 webhook 或应用凭证
- 需要的权限：发送消息权限

### 2. 执行步骤

**方式一：发送文本消息**

```
发送企业微信消息：

接收人：产品群
内容：
📋 今日站会提醒

时间：10:00
地点：会议室 A
议程：
1. 昨日进展
2. 今日计划
3. 阻塞问题

请准时参加！
```

**方式二：发送卡片消息**

```
发送企业微信卡片：

接收人：开发群
卡片类型：通知卡片
标题：🔔 系统部署完成
内容：
- 环境：生产环境
- 版本：v1.2.3
- 时间：2024-03-24 15:30
- 状态：成功

按钮：
- [查看详情] -> https://example.com/deploy/123
- [回滚] -> https://example.com/rollback
```

**方式三：发送文件**

```
发送文件到企业微信：

接收人：财务组
文件：/reports/月度财务报告.xlsx
说明：2024年3月财务报告，请查收
```

**方式四：定时消息**

```
设置定时企业微信消息：

接收人：全员群
时间：每天 09:00
内容：
🌞 早安提醒

今日待办：
- [ ] 查看邮件
- [ ] 检查日程
- [ ] 更新任务状态

祝大家工作愉快！
```

**方式五：告警通知**

```
发送告警到企业微信：

接收人：运维群
告警级别：严重
标题：⚠️ 服务器 CPU 过高
内容：
- 服务器：prod-server-01
- CPU 使用率：95%
- 时间：2024-03-24 14:23
- 持续时间：5 分钟

建议立即处理！
```

**OpenClaw 会自动：**
1. 连接企业微信 API
2. 发送消息或文件
3. 返回发送状态
4. 处理失败重试

### 3. 效果展示

**消息发送成功：**
```

接收人：产品群（15 人）
消息类型：文本
发送时间：2024-03-24 09:30:00
消息 ID：msg_20240324001

状态：已送达 ✓
```

**卡片消息展示：**
```
┌─────────────────────────┐
│ 🔔 系统部署完成         │
├─────────────────────────┤
│ 环境：生产环境          │
│ 版本：v1.2.3            │
│ 时间：2024-03-24 15:30  │
│ 状态：成功              │
├─────────────────────────┤
│ [查看详情]  [回滚]      │
└─────────────────────────┘
```

## 需要准备什么

| 依赖项 | 说明 | 获取方式 |
|--------|------|----------|
| OpenClaw | >= 1.0.0 | 官方安装 |
| 企业微信机器人 | Webhook URL | 群设置中添加 |
| 或应用凭证 | CorpID + Secret | 企业微信管理后台 |
