# Slack 消息发送

## 干什么
让 OpenClaw 自动发送 Slack 消息，支持频道消息、私信、文件分享、富文本格式。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的配置：Slack Bot Token 或 Incoming Webhook
- 需要的权限：chat:write, files:write

### 2. 执行步骤

**方式一：发送频道消息**

```
发送 Slack 消息：

频道：#general
内容：
🚀 新版本已发布！

版本：v2.0.0
更新内容：
• 新增暗黑模式
• 优化性能 50%
• 修复 15 个 bug

完整更新日志：https://example.com/changelog
```

**方式二：发送私信**

```
发送 Slack 私信：

用户：@john
内容：
Hi John，你的 PR #123 已审核通过。
请查看：https://github.com/repo/pull/123
```

**方式三：发送富文本消息**

```
发送 Slack 富文本消息：

频道：#dev-team
内容：
*重要通知* 🔔

代码审查会议已安排：

> 时间：2024-03-25 14:00
> 地点：会议室 B
> 议题：API 重构方案

请提前准备：
• 阅读 _设计文档_
• 查看代码变更
• 准备问题和建议

_此消息由 OpenClaw 自动发送_
```

**方式四：发送文件**

```
发送文件到 Slack：

频道：#reports
文件：/reports/月度报告.pdf
标题：2024年3月月度报告
说明：本月销售数据和分析报告
```

**方式五：交互式消息**

```
发送交互式 Slack 消息：

频道：#team-updates
标题：部署确认
内容：新版本 v1.5.0 已准备就绪，是否部署到生产环境？
按钮：
- [立即部署] -> action:deploy
- [取消] -> action:cancel
- [查看详情] -> https://example.com/deploy
```

**方式六：定时提醒**

```
设置 Slack 定时提醒：

频道：#standup
时间：每天 09:30
内容：
📢 站会提醒

请在频道更新：
1. 昨天完成了什么
2. 今天计划做什么
3. 有什么阻塞

格式：使用 `/standup` 命令快速提交
```

**OpenClaw 会自动：**
1. 连接 Slack API
2. 格式化消息内容
3. 发送消息或文件
4. 返回发送状态

## 需要准备什么

| 依赖项 | 说明 | 获取方式 |
|--------|------|----------|
| OpenClaw | >= 1.0.0 | 官方安装 |
| Slack Bot Token | xoxb-xxx | Slack App 设置 |
| 或 Webhook URL | Incoming Webhook | Slack 频道设置 |
