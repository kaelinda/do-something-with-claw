# 数据可视化

## 干什么
让 OpenClaw 将数据转换为可视化图表，支持折线图、柱状图、饼图、散点图等多种类型。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的技能：无
- 数据格式：CSV、JSON、表格

### 2. 执行步骤

**方式一：从数据生成图表**

```
将以下销售数据可视化：

月份,销售额,订单数
2024-01,125000,450
2024-02,138000,520
2024-03,142000,485
2024-04,156000,510

要求：
- 生成柱状图展示销售额
- 生成折线图展示趋势
- 添加标题和图例
```

**方式二：多维度对比**

```
创建对比图表：

数据：
产品 A：Q1=100, Q2=120, Q3=115, Q4=140
产品 B：Q1=80, Q2=95, Q3=110, Q4=125
产品 C：Q1=60, Q2=75, Q3=85, Q4=95

要求：
- 分组柱状图对比三个产品
- 每季度一组
- 不同颜色区分产品
```

**方式三：数据分布可视化**

```
可视化用户年龄分布：

数据：
18-24岁：1200人
25-34岁：3500人
35-44岁：2800人
45-54岁：1500人
55岁以上：600人

要求：
- 饼图展示占比
- 标注百分比
- 突出最大群体
```

**方式四：趋势分析图**

```
生成网站流量趋势图：

时间序列数据：
- 过去 30 天的日访问量
- 过去 30 天的转化率
- 过去 30 天的跳出率

要求：
- 双 Y 轴图表
- 左轴：访问量（柱状图）
- 右轴：转化率、跳出率（折线图）
- 标注关键事件点
```

**方式五：从文件读取数据**

```
从 CSV 文件生成图表：

文件：/data/monthly_report.csv
列：日期、收入、支出、利润

生成：
1. 收入 vs 支出对比图
2. 利润趋势图
3. 月度环比增长图
```

**方式六：交互式仪表盘**

```
创建数据仪表盘：

数据源：销售数据库
维度：
- 时间：按月/季度/年
- 地区：按省份/城市
- 产品：按类别
- 渠道：线上/线下

图表组合：
1. 总销售额大数字卡片
2. 月度趋势折线图
3. 地区分布地图
4. 产品类别饼图
5. 渠道对比柱状图
```

**OpenClaw 会自动：**
1. 解析数据格式
2. 推荐合适的图表类型
3. 生成可视化图表
4. 添加标注和图例
5. 输出为图片或 HTML

### 3. 效果展示

**柱状图输出：**
```
销售额月度对比

销售额

150K ┤         ████ ████
140K ┤    ████ ████ ████
130K ┤███ ████ ████ ████ ████
120K ┤███ ████ ████ ████ ████
     └────────────────────
       1月  2月  3月  4月

数据标注：
1月：125,000
2月：138,000 (+10.4%)
3月：142,000 (+2.9%)
4月：156,000 (+9.9%)
```

**饼图输出：**
```
用户年龄分布

        25-34岁 (43%)
         ╱─────────╲
       ╱             ╲
     ╱   35-44岁     ╲
```

**数据洞察：**
```
📈 数据分析洞察

关键发现：
1. 销售额持续增长，4月达到峰值 156,000
2. 月均增长率：8.1%
3. 2月增长最显著（+10.4%）

趋势预测：
- 按当前增速，5月预计：170,000
- 建议关注 2 月增长原因，复制成功经验

异常点：
- 3 月增长率下降至 2.9%，需分析原因
```
