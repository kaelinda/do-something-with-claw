# SEO 优化内容

## 干什么
让 OpenClaw 帮你生成 SEO 友好的内容，包括关键词优化、元描述、标题优化、内容结构化。

## 怎么干

### 1. 准备工作
- 需要 OpenClaw 版本：>= 1.0.0
- 需要的技能：无
- 需要的配置：目标关键词列表

### 2. 执行步骤

**方式一：生成 SEO 标题和描述**

```
为以下文章生成 SEO 优化内容：

主题：AI 助手提升工作效率
目标关键词：AI 助手、效率提升、自动化工作流

生成：
1. 5 个标题选项（包含关键词）
2. Meta 描述（150-160 字符）
3. H2/H3 大纲建议
```

**方式二：内容 SEO 优化**

```
优化以下内容的 SEO：

原文：
[paste your content]

目标关键词：OpenClaw、自动化、AI 助手
要求：
- 关键词密度 2-3%
- 自然融入关键词
- 添加相关长尾词
- 优化段落结构
```

**方式三：关键词研究**

```
帮我研究关键词"AI 助手"：
- 相关长尾关键词
- 搜索意图分析
- 竞争难度评估
- 内容建议
```

**方式四：批量生成 SEO 内容**

```
为以下关键词批量生成 SEO 文章大纲：

关键词列表：
1. 自动化工作流
2. AI 生产力工具
3. 智能助手应用

每个关键词生成：
- 标题
- Meta 描述
- 文章大纲（H2/H3）
- 关键要点
```

**OpenClaw 会自动：**
1. 分析目标关键词和搜索意图
2. 生成优化的标题和描述
3. 创建结构化内容大纲
4. 确保关键词自然分布
5. 提供优化建议
